# 排列3 (Pick3) 预测技能 — CHANGELOG

## V1.0.0 (2026-06-24)

### 初始发布
基于 DLT 大乐透预测系统架构，针对排列3专项重构。

**核心架构:**
- 位置采样器 (P3PositionSampler): 百位/十位/个位独立频率分析，冷热/趋势/均衡四策略
- 模式识别器 (P3PatternRecognizer): 和值/跨度/奇偶比/大小比/组类型/012路/跨位重复等9种特征
- 博弈论分析器 (P3GameTheoryAnalyzer): 期望值排序、纳什均衡概率
- 数学过滤器 (P3MathFilter): 和值/跨度/奇偶/组类型过滤
- 约束引擎 (P3ConstraintEngine): 硬约束+软约束+3种预设策略

**预测能力:**
- 直选推荐 (精确顺序定位)
- 组三推荐 (2个相同数字)
- 组六推荐 (3个不同数字)
- 直选复式/组选复式方案

**数据同步:**
- 体彩数据API (gameNo=53) 自动获取排列3开奖数据
- 500彩票网 HTML 解析备选回退
- 自动追加新期号到 Excel

**存储:**
- prediction_store: 自动保存 + 保留最近10期
- 回测对比支持（直选/组三/组六分别统计）

**与 DLT 差异:**
- 前区5+后区2 → 3位数字(每位0-9)
- 六池+遗传神经网络 → 位置采样+模式匹配+博弈论
- 复式12种 → 直选复式(定位) + 组选复式
- 号码空间 2142万 → 1000
- 无深度学习(CPU友好)
