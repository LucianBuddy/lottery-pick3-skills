# 排列3 (Pick3) 预测技能 V2.8.0

全量枚举(1000/90/120) + 置换检验MI权重校准 + 位置级转移矩阵 + IRL在线学习。NN/1000×1000转移矩阵已移除。数据自动同步，支持直选/组三/组六/复式投注。

---

## 数据同步

调用 `predict()` 时自动触发。体彩API(gameNo=53)优先 → 500彩票网回退，新期号追加到 Excel。

## 核心调用

```python
from p3_fusion_complete import Pick3FusionComplete

fusion = Pick3FusionComplete()
result = fusion.predict()
print(result['zx_bets'])     # 直选推荐 (10注)
print(result['z3_bets'])     # 组三推荐 (5注)
print(result['z6_bets'])     # 组六推荐 (5注)
print(result['compound_bets'])  # 复式方案

bt = fusion.backtest(n_periods=100)  # 回测
info = fusion.info()  # 技能信息
```

或者 CLI：
```bash
python3 p3_lottery_skill.py predict          # 预测
python3 p3_lottery_skill.py backtest         # 回测
python3 p3_lottery_skill.py info             # 技能信息
```

## 架构快速参考

- **全量枚举**：直选1000/组三90/组六120，全量评分后排序选择
- **L1位置频率**（权重ZX=0.45/Z3=0.35/Z6=0.40）：多窗口频率×0.5 + 转移概率×0.3 + 泊松反常×0.2
- **L2模式匹配**（权重ZX=0.25/Z3=0.30/Z6=0.25）：和值+跨度+重号（70%）+ 跨期模式识别（30%）
- **L3数学合理性**（权重ZX=0.20/Z3=0.25/Z6=0.25）：奇偶比（50%）+ 数学特征评分（50%）
- **L4博弈论**（权重0.10）：历史频率期望值
- **约束引擎**：硬约束(和值/跨度/组类型) + 软约束(奇偶/012路/和值区间)
- **多样性选择**：和值+跨度双维度分桶轮询
- **回测路径**：完全复用predict评分逻辑(_compute_layers)，无代码重复

## 与DLT技能的区别

| 维度 | 大乐透 | 排列3 |
|------|--------|-------|
| 选号规则 | 5+2 (35/12选) | 3位 (0-9×3) |
| 号码空间 | 2142万 | 1000 |
| 预测方向 | 六池+遗传+神经网络 | 位置采样+模式匹配+博弈论 |
| 投注类型 | 单式/复式/胆拖 | 直选/组三/组六/复式 |
| 计算负载 | 高(PyTorch CPU >120s) | 低(<10s) |

## 性能注意事项

- 纯 CPU 低负载，无需 GPU
- 内存 <200MB 即可运行
- 回测50期约需30-60秒
