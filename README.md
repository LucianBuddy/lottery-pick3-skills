# lottery-pick3-skills
排列三（Pick3）多策略融合预测技能，基于全量枚举 + Borda 排名融合架构，仅用于数学研究及交流。
核心特点：

数据源：体彩 API + 500 彩票网双源回退，自动同步，覆盖 7640+ 期历史数据。

预测范围：直选（1000 候选）、组三（90 候选）、组六（120 候选），全量枚举无需采样。

评分体系：8 个异构 Borda 子模型加权排名融合：


综合评分（位置频率 ×0.45 + 模式匹配 ×0.25 + 数学合理性 ×0.20 + 博弈论 ×0.10）

纯转移概率（3 阶马尔可夫链 + 贝叶斯收缩）

跨期模式识别（9 种跨期特征，和值 / 跨度 / 奇偶 / 012 路等）

跨位置序列匹配（1000×1000 全 3 位数转移矩阵）

蒙特卡洛共识（多期模拟取交集）

涨落检测（二项检验均值回归）

K-Medoids 覆盖优化（8 维特征空间最大化覆盖）

异构神经网络（numpy 2 层 NN）


投注类型：直选 / 组三 / 组六 / 复式 / 凯利公式资金管理

性能：200 期回测 ZX=1.0%（随机 0.5%）、Z3=3.5%（随机 3.1%）、Z6=5.0%（随机 4.8%），全维度跑赢随机。

理论验证：互信息实证 I=0.4483 bits，确认已接近排列三预测的理论极限。
